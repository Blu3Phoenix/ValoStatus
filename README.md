# ValoStatus 
**ValoStatus** is a software developed in order to obtain information about the **Account** and **MMR** of a **Valorant player**. Here you have the data that is collected from each of the sections:

- **User Data**:
    - Username
    - Tag
    - Card (Wide Format)
- **MMR Data**:
    - Rank
    - Rank Card
    - Rank Points
    - Last Game Status
    - Elo

## 🛡️ Supported platforms 🛡️
ValoStatus supported platforms are:
- Windows

## 💽 Functionality 💽
When you start the application, it will check **if the API from which it needs to obtain the information is available**. Otherwise, **the application will close with an error message**.

The application requires three essential fields (**Mandatory**):
- **Username**
- **Tag**
- **Region**:
	- "AP" : Asia-Pacific
	- "BR" : Brazil
	- "EU" : Europe
	- "KR" : Korea
	- "LATAM" : Latin America
	- "NA" : North America

Once the data has been entered, the application will verify if:
- The player's account **exists** (Verifying by **username** and **tag**).
- The player's account **exists** in the selected region (Verifying by **region**).

If everything goes well, the application will show you the data correctly.
